# HR Salary & Promotion Analysis

An end-to-end Data Analysis project on IBM HR Analytics dataset (1,470 employees) focusing on salary benchmarking and promotion patterns.

## Tech Stack
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Plotly Dash (Interactive Dashboard)
- Jupyter Notebook

## Key Findings
- Job Level and Monthly Income have 0.95 correlation
- Performance Rating 4 employees receive 20-24% hike vs 12-19% for Rating 3
- Managers earn 6.5x more than Sales Representatives
- Sales department has highest avg promotion gap (~2.2 years)

python app.py
